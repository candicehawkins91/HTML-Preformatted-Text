Understanding the pre Tag in HTML

What is the pre element?

The pre element in HTML is used to display text in a pre-formatted way. This means that spaces, tabs, and line breaks are preserved exactly as written, unlike normal text rendering where browsers collapse whitespace.

Example of using pre:

The 
 element preserves:
  - Line breaks
  - Extra spaces
  - Indentation

This makes it useful for displaying code, poetry, or different art forms that uses text.

Formatting Example with JavaScript

Here is some JavaScript shown inside a normal paragraph:

function greet() { console.log("Hello, world!"); }

Here is the same JavaScript shown inside a pre tag:

function greet() {
    console.log("Hello, world!");
}
